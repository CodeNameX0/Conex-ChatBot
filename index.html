✨ 주요 기능

🚀 초고속 응답

Groq의 LPU 기술로 매우 빠른 AI 응답 속도를 제공합니다.

💰 무료 사용

월 1만 토큰 무료 제공으로 부담 없이 사용할 수 있습니다.

🎨 모던 UI

깔끔한 다크 테마와 말풍선 스타일의 직관적인 인터페이스입니다.

📱 모바일 최적화

모든 디바이스에서 완벽한 사용자 경험을 제공합니다.

🔐 보안 강화

API 키를 안전하게 관리하여 GitHub에 올려도 안전합니다.

🌐 오픈소스

MIT 라이선스로 자유롭게 사용하고 수정할 수 있습니다.

🚀 빠른 시작

1

레포지토리 클론

GitHub에서 프로젝트를 다운로드합니다.

git clone https://github.com/CodeNameX0/Conex-ChatBot.git
2

패키지 설치

필요한 Python 패키지들을 설치합니다.

py -m pip install -r requirements.txt
3

API 키 설정

Groq Console에서 무료 API 키를 받아 설정합니다.

$env:GROQ_API_KEY="your_api_key_here"
4

앱 실행

Flask 서버를 시작하고 브라우저에서 접속합니다.

py app.py
